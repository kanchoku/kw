■  このパッケージについて

    「漢直Win」は、徳岡宏樹さんの作成されたソフトです。
    T-Code メーリングリストのメンバーによって、機能が追加され
    バージョン 1.26eb としてパッケージ化されています。

    今回バージョン 1.26eb をもとに一部機能強化を行い、また
    新しい機能に対応したマニュアルを追加して、バージョン 1.28 として
    再パッケージ化を行いました。

    現在は GitHub 上でメンテナンスされています。



■ 使用法

    doc フォルダ の中の kanchoku.html をご覧ください。



■  ファイル

      readme.txt        このファイル
      news.txt          1.27〜 版の変更点
      news127.txt       1.27 版での主な変更点
      history.txt       履歴

      doc/kanchoku.html 「漢直Win 1.28」のマニュアル

      old/kanchoku.txt  1.26 版 (オリジナル版) のドキュメント

    ソースファイル

    詳細は、各ファイルを直接ご覧ください。
